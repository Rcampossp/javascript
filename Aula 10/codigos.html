"Somando valores em JS"

Primeira parte dentro do HTML

Para efetuarmos soma de valores dentro do Jscript, devemos primeiramente iniciarmos com as tag
input.Esta primeira parte é inserira/criada dentro do HTML.

<input type = number" name = "..." id = ".....">
que vai formar um campo como este: Onde poderemos inserir valores.

+

Criaremos outro campo pois iremos trabalhar com soma de valores.

Vamos Utilizar outro input como "Botão"
<input type = button" value = "somar" onclick="somar()">.

Agora Criaremos uma div com Id (res)para receber o resultado.
<div id = "res">RESULTADO</div>

Resultado

Nossa primeira parte no HTML fica assim, vamos agora para outa parte que é a criação das variaveis dentro do Jscript
